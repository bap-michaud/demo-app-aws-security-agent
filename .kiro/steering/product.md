# Product Overview

This is a demo application showcasing AWS security agent capabilities using the Model Context Protocol (MCP).

## Purpose

Demonstrate AWS security operations and IAM management through an AI-powered agent interface, leveraging MCP servers for AWS documentation and IAM operations.

## Key Features

- AWS IAM user and role management
- Security policy analysis and recommendations
- Integration with AWS documentation for context-aware assistance
- MCP-based architecture for extensible AWS operations
